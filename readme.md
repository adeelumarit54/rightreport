# RightReport Word Add-in

**RightReport** is a Microsoft Word Add-in that allows users to view and insert inspection report data directly into a Word document.  
It uses React + TypeScript for the frontend and integrates with the Word JavaScript API (Office.js).

---

## üì¶ Features

- View inspection reports in a modern React UI (with Fluent UI / MUI design).
- Preview and expand/collapse long inspection details.
- Insert report fields (text or images) into the active Word document.
- Automatically handle formatting such as bullet points and bold text.
